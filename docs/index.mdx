## English

### ✨ Features

- **🔒 Local Models** — Work with Ollama models locally without sending data to the cloud. Complete privacy and control.
- **💻 CLI Interface** — Powerful terminal interface built with Ink (React) for comfortable work directly from the command line.
- **🌐 Web UI** — Modern web interface on Next.js with Monaco Editor and built-in terminal.
- **🔌 Plugin System** — Extend functionality with plugins. Dynamic loading and sandbox security.
- **🛡️ Security Sandbox** — Plugin isolation with trust levels. Control access to files, network, and commands.
- **⚡ Context Caching** — KV-cache reuse for faster repeated requests. Save tokens and time.
- **↩️ Undo/Redo** — Command Pattern for undoing and redoing operations. Safe code editing.
- **📡 MCP Support** — Integration with Model Context Protocol for extended capabilities.

## Русский

### ✨ Возможности

- **🔒 Локальные модели** — Работайте с Ollama моделями локально без отправки данных в облако. Полная приватность и контроль.
- **💻 CLI интерфейс** — Мощный терминальный интерфейс на Ink (React) для комфортной работы прямо из командной строки.
- **🌐 Web UI** — Современный веб-интерфейс на Next.js с Monaco Editor и встроенным терминалом.
- **🔌 Plugin System** — Расширяйте функциональность с помощью плагинов. Динамическая загрузка и sandbox безопасность.
- **🛡️ Security Sandbox** — Изоляция плагинов с уровнями доверия. Контроль доступа к файлам, сети и командам.
- **⚡ Context Caching** — KV-cache reuse для ускорения повторных запросов. Экономия токенов и времени.
- **↩️ Undo/Redo** — Command Pattern для отмены и повтора операций. Безопасное редактирование кода.
- **📡 MCP Support** — Интеграция с Model Context Protocol для расширенных возможностей.
