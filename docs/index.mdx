## 🚀 Quick Start

```bash
# Install via npm
npm install -g @ollama-code/ollama-code

# Or via pnpm
pnpm install -g @ollama-code/ollama-code

# Run
ollama-code
```

### Prerequisites

1. **Ollama** installed and running ([ollama.ai](https://ollama.ai))
2. **Node.js** 18+ installed
3. A compatible model pulled (e.g., `ollama pull qwen2.5-coder:7b`)

## 🛠️ 22+ Built-in Tools

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(auto-fit, minmax(200px, 1fr))', gap: '1rem', margin: '1rem 0' }}>

**File System:**
- `read_file` - Read file contents
- `write_file` - Create/overwrite files
- `edit` - Edit existing files
- `glob` - Find files by pattern
- `list_directory` - List directory contents

**Development:**
- `python_dev` - Python execution
- `nodejs_dev` - Node.js execution
- `golang_dev` - Go development
- `rust_dev` - Rust development
- `java_dev` - Java development

**Search:**
- `grep_search` - Search in files
- `web_search` - Search the web
- `web_fetch` - Fetch web content

**Shell:**
- `run_shell_command` - Execute commands
- `bash` - Run bash scripts

**AI:**
- `task` - Spawn sub-agents
- `skill` - Use specialized skills
- `todo_write` - Manage tasks
