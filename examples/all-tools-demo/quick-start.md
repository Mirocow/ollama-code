# Quick Start: Essential Tools Demo

Minimal example covering the most commonly used tools.

## Prompt

```
Help me with these basic tasks using ollama-code tools:

1. Show me the current time using `timestamp`
2. List files in the current directory using `list_directory`
3. Read the README file using `read_file`
4. Create a new file called "hello.txt" with content "Hello, World!" using `write_file`
5. Run a simple shell command using `run_shell_command`
6. Show git status using `git_smart`
7. Create a simple todo list using `todo_write`
```

## Expected Tools

| Tool | Purpose |
|------|---------|
| timestamp | Get current time |
| list_directory | Browse files |
| read_file | Read content |
| write_file | Create file |
| run_shell_command | Execute shell |
| git_smart | Git operations |
| todo_write | Task tracking |
