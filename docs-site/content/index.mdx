## ✨ Features

### 🔒 Local Models
Work with Ollama models locally without sending data to the cloud. Complete privacy and control over your code.

### 💻 CLI Interface
Powerful terminal interface built with Ink (React) for comfortable work directly from the command line.

### 🌐 Web UI
Modern web interface on Next.js with Monaco Editor and built-in terminal.

### 🔌 Plugin System
Extend functionality with plugins. Dynamic loading and sandbox security for safe extensibility.

### 🛡️ Security Sandbox
Plugin isolation with trust levels. Control access to files, network, and commands.

### ⚡ Context Caching
KV-cache reuse for faster repeated requests. Save tokens and reduce response time.

### ↩️ Undo/Redo
Command Pattern for undoing and redoing operations. Safe code editing with history.

### 📡 MCP Support
Integration with Model Context Protocol for extended capabilities and tool integration.

## 🚀 Quick Start

```bash
# Install via npm
npm install -g @ollama-code/ollama-code

# Or via pnpm
pnpm install -g @ollama-code/ollama-code

# Run
ollama-code
```

### Prerequisites

1. **Ollama** installed and running ([ollama.ai](https://ollama.ai))
2. **Node.js** 18+ installed
3. A compatible model pulled (e.g., `ollama pull qwen2.5-coder:7b`)
