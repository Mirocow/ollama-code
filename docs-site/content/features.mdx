# Features

Explore all the powerful features of Ollama Code.

## 🆕 Latest Release (v0.17.0)

### Git Workflow Tool
Complete git workflow integration with support for GitHub and GitLab (including self-hosted):
- **Basic operations**: status, add, commit, push, pull, fetch
- **Branch operations**: create, switch, merge
- **MR/PR creation**: Auto-detect GitHub/GitLab and create merge requests
- **Authentication**: Check status, login instructions, token management
- **Clone repositories**: With branch, depth, and recursive options

### Platform Support
- **GitHub**: Full support via `gh` CLI
- **GitLab.com**: Full support via `glab` CLI
- **Self-hosted GitLab**: Auto-detection and support
- **Fallback**: Instructions provided when CLI tools not installed

## Local Models

Work with Ollama models locally. No API keys required. Works offline with complete privacy.

## CLI Interface

Beautiful terminal UI built with Ink (React). Real-time streaming, syntax highlighting, and keyboard shortcuts.

## Web UI

Modern web interface with Monaco Editor, built-in terminal, and chat interface.

## Built-in Tools

### File System
Read, write, and edit files. Find files by pattern. List directories.

### Development
Execute Python, Node.js, Go, Rust, and Java code.

### Git Operations
Complete git workflow: commit, push, pull, create branches, and MR/PR creation. Advanced operations: stash, cherry-pick, rebase, bisect.

### Search
Search within files. Search the web. Fetch web content.

### AI
Spawn sub-agents. Use specialized skills. Manage task lists.

## Plugin System

Extend functionality with plugins. Dynamic loading and sandboxed execution with trust levels.

## MCP Support

Integration with Model Context Protocol for external tools and resources.

## Undo/Redo

Safe code editing with full history and checkpoint system.

## Context Caching

KV-cache reuse for faster repeated requests. Save tokens and reduce response time.
